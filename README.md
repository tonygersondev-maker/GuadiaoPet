ONG Animal — Plataforma Web Completa

Documentação Oficial do Projeto
Este repositório contém o desenvolvimento de uma plataforma web profissional destinada a ONGs de proteção animal, permitindo gerenciamento de atividades, divulgação de projetos, captação de recursos, engajamento de voluntários e interação com doadores e visitantes.

-->> Objetivo Geral <<--
Desenvolver uma plataforma web completa que possibilite a ONGs gerenciar suas operações, apresentar seus projetos, captar doações, engajar voluntários e oferecer transparência ao público.

-->> Objetivos Específicos
Aplicar HTML5 para criar estruturas semânticas completas e acessíveis.

Implementar CSS3 avançado com design system, responsividade e componentes visuais.

Desenvolver funcionalidades dinâmicas usando JavaScript.

Integrar boas práticas de acessibilidade, infraestrutura e versionamento.

Reproduzir um ambiente profissional com metodologias ágeis.

Documentar cada funcionalidade e requisito técnico da plataforma.
